# dot-files

### Tools i used to use in the daily life tasks.

- nvim
- tmux
- i3w
- zshrc

### Setup

```bash
# Nvim (symlink)
ln -s ~/workspace/github/dot-files/nvim ~/.config/nvim

# Tmux (symlink)
ln -s ~/workspace/github/dot-files/tmux.conf ~/.config/tmux/tmux.conf

# Zsh
ln -s ~/workspace/github/dot-files/.zshrc ~/.zshrc
```
